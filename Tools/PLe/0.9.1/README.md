HotKeys:<br>
[F2] Menu<br>
[F10] Exit<br>
[TAB] Switch Tree/View<br>
[Enter] Select View<br>

View:<br>
[PgDn] Page down
[PgUp] Page up
[Home] Begin of dump
[End] End of dump

Available Dumps:<br>
- ACPI
- PCIe
- SMBIOS
- Memory
- UEFI
